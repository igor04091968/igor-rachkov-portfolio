# Откуда взялись скриншоты

Скриншоты созданы автоматически в Playwright 21 июля 2026 года. Во время съёмки
приложение не подключалось к действующей АТС.

- в качестве SSH-цели использован недоступный локальный адрес;
- все ответы API подставлялись непосредственно в браузере;
- реальные номера, имена, адреса и конфигурации не использовались;
- размер снимков рабочего экрана — 1600×1000;
- размер мобильного снимка — 430×932.

Документационные IP-адреса на отдельных кадрах принадлежат RFC 5737.
